# MoneyAgent 前端官网 — 设计文档

## 1. 概述

### 1.1 目标

为 MoneyAgent 构建一个 NVIDIA 风格的科技官网，包含 Landing 展示页、实时数据仪表盘、配置管理、日志历史四个页面。

### 1.2 设计参考

NVIDIA 官网 (nvidia.com) — 暗黑主题、粒子动画、绿色光效、硬核科技感

### 1.3 技术栈

| 层 | 选择 |
|---|------|
| 框架 | React 18 + TypeScript |
| 样式 | Tailwind CSS 3 |
| 动画 | Framer Motion |
| 图表 | Recharts |
| 3D/粒子 | Three.js + @react-three/fiber |
| 构建 | Vite |
| API | Python Flask (新增后端API) |
| 部署 | 同一服务器，Python serve React 静态文件 |

## 3. 页面设计

### 3.1 首页 Landing

**Hero 区域:**
- 粒子背景 (Canvas / Three.js 粒子网络)
- 大标题 "你的AI合伙人，会自己赚钱" (打字机动画)
- 副标题 "自进化 · 全自动 · 24/7赚钱"
- 3D 旋转的 Agent 图标
- CTA 按钮 "启动你的Agent" (绿色渐变 + 脉冲光晕)

**数据跑马灯:**
- 实时滚动显示: 本月收入 / 进化次数 / 赚钱渠道数 / 运行天数
- 数字跳动动画 (count-up effect)

**能力卡片 (4张 Glassmorphism):**
1. 🧬 自进化 — Agent 自己读代码、写代码、测试、提交
2. 💰 自赚钱 — 自动发现平台、注册账号、运营变现
3. 🛡️ 安全第一 — 四层防自毁，隐私永不泄露
4. 📊 透明公开 — 每笔收支记录在链(账本)

**如何工作 (三步流程图):**
1. 配置 Token → 2. 启动 Agent → 3. 坐等收钱

**顶部卡片行 (4张):**
- 本月收入 ($ + 数字跳动)

## 4. 数据流

### 4.1 API 设计

Python 后端新增 Flask API (端口 5000):

```
GET  /api/status        — Agent状态 (运行中/休眠/停止)
GET  /api/ledger        — 账本数据 (本月汇总+明细)
GET  /api/ledger/month  — 月度历史
GET  /api/channels      — 各渠道状态和收入
GET  /api/evolution     — 进化日志 (最近N条)
GET  /api/notifications — 通知列表
GET  /api/resources     — 服务器资源状态

POST /api/config        — 更新配置
POST /api/channels/:id/toggle — 开关渠道
POST /api/agent/evolve  — 触发立即进化
POST /api/agent/pause   — 暂停Agent
POST /api/agent/resume  — 恢复Agent

PUT  /api/payment/:id   — 审批支付请求
```

### 4.2 前端数据获取

- 仪表盘页面: 每10秒轮询 `/api/status` + `/api/ledger`
- 日志页面: 按需加载 + 分页
- 配置页面: 读取后本地缓存，提交时写回

### 5.1 部署架构

```
用户浏览器 → :80 (nginx) → React静态文件 (frontend/dist/)
                         → /api/* → Flask :5000 → Agent data/
```

开发时: `npm run dev` (Vite :3000) + `python server/api_server.py` (:5000)
生产时: `npm run build` → Flask serve 静态文件 + API 同端口

## 6. 动画规范

| 场景 | 动画 | 时长 | 缓动 |
|------|------|------|------|
| 页面切换 | 淡入 + 上移 20px | 0.4s | ease-out |
| 卡片出现 | 依次淡入 (stagger 0.1s) | 0.5s | ease-out |
| 数字跳动 | count-up 数字滚动 | 1.5s | ease-out |
| 悬浮 | 卡片上浮 4px + 阴影增强 | 0.3s | ease |
| 按钮 | hover 发光 + 点击涟漪 | 0.2s / 0.6s | ease |
| 进度条 | 宽度过渡 + 绿色流光 | 1s | ease-out |
| 粒子背景 | 持续60fps Canvas动画 | 持续 | - |
| 时间轴 | 节点依次出现 (stagger) | 0.3s | ease-out |
| Toast | 右侧滑入 + 自动消失 | 0.3s / 3s | spring |
| 开关 | 滑块位移 | 0.2s | ease |
